My Journey through HTML, CSS, Git, and Bootstrap

1st March, 2024 by Akech Sandra

Categories

  • HTML
  • CSS
  • GIT
  • BOOTSTRAP
Entering the world of software engineering can feel like stepping into a vast, unknown territory. However, my experience in an introduction to software engineering class has been nothing short of enlightening, as I delved into the foundational elements of web development: HTML, CSS, Git, and Bootstrap.
Introduction to HTML

The class began with a deep dive into HTML, the backbone of any web page. Learning how to structure content using tags and elements was like learning a new language, but with practice, I began to see the beauty in its simplicity and power to create meaningful web pages.

Introduction to CSS

Next, we explored CSS, the style sheet language that brings HTML to life. Understanding how to use CSS to add colors, fonts, and layouts to our web pages was a game-changer. Suddenly, the static HTML pages we created came to life with vibrant designs and appealing layouts.

Introduction to Git

One of the most valuable lessons I learned was the importance of version control with Git. Learning how to track changes, collaborate with others, and revert to previous versions gave me a sense of security and control over my projects that I had never experienced before.

Introduction to Bootstrap

Finally, we delved into Bootstrap, a front-end framework that streamlines the process of designing responsive and mobile-first websites. Using Bootstrap's pre-designed components and grid system, I was able to create professional-looking websites with ease, without having to write extensive CSS code from scratch.
